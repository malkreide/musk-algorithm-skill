# Zum Musk-Algorithmus Skill beitragen

Vielen Dank für dein Interesse, zu diesem Projekt beizutragen!

## Wie du beitragen kannst

### Probleme melden

1. Überprüfe, ob das Problem bereits im Bereich [Issues](https://github.com/malkreide/musk-algorithm-skill/issues) existiert
2. Falls nicht, erstelle ein neues Issue mit einem klaren Titel und einer Beschreibung
3. Füge Beispiele für das problematische Verhalten hinzu, falls zutreffend

### Änderungen einreichen

1. **Forke das Repository** auf dein eigenes GitHub-Konto
2. **Erstelle einen neuen Branch** für dein Feature oder deinen Bugfix:
   ```bash
   git checkout -b feature/dein-feature-name
   ```
3. **Nimm deine Änderungen vor** mit klaren, beschreibenden Commit-Nachrichten
4. **Teste deine Änderungen** gründlich
5. **Reiche einen Pull Request ein** mit:
   - Klarer Beschreibung dessen, was du geändert hast und warum
   - Verweis auf verwandte Issues
   - Beispiele für die Verbesserung, falls zutreffend

### Bereiche für Beiträge

Wir sind besonders interessiert an Beiträgen in diesen Bereichen:

- **Praxisbeispiele** aus der öffentlichen Verwaltung oder dem Bildungswesen
- **Domänenspezifische Anpassungen** (z.B. für Gesundheitswesen, Justiz, Sozialwesen)
- **Mehrsprachige Unterstützung** (Französisch, Italienisch, Englisch)
- **Evaluationsmetriken** zur Messung der Optimierungswirkung
- **Integrationsleitfäden** für verschiedene Plattformen
- **Dokumentationsverbesserungen**

### Richtlinien

- Folge dem bestehenden Dokumentationsformat
- Verwende Schweizer Rechtschreibung (kein ß, ss statt ß)
- Halte Änderungen fokussiert und atomar
- Schreibe klare Commit-Nachrichten
- Aktualisiere die Dokumentation entsprechend deinen Änderungen
- Sei respektvoll und konstruktiv in Diskussionen

### Fragen?

- Öffne eine [Diskussion](https://github.com/malkreide/musk-algorithm-skill/discussions)
- Kontaktiere über Issues
- Prüfe zuerst die bestehende Dokumentation

## Verhaltenskodex

Wir verpflichten uns, eine einladende und inklusive Umgebung zu schaffen. Bitte:

- Sei respektvoll und rücksichtsvoll
- Heisse Neuankömmlinge willkommen und hilf ihnen beim Einstieg
- Konzentriere dich auf das, was am besten für die Community ist
- Zeige Empathie gegenüber anderen Mitwirkenden

## Lizenz

Durch deinen Beitrag stimmst du zu, dass deine Beiträge unter der MIT-Lizenz lizenziert werden.
